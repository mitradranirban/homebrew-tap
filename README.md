# homebrew-tap
Homebrew cascs for my apps such as [Offed](https://fonts.atipra.in/offed.html) and [ColrPak](https://github.com/mitadranirban/colr-pak) for easy installation and upgrade in macOS

To install the apps through homebrew in macOS, open a terminal in your mac and run these commands
### 1. Add this repository to your Homebrew
`brew tap mitradranirban/tap`

#### 2. Install the apps
```
brew install --cask colr-pak
brew install --cask offed
```
